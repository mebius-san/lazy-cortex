## Highlights since `5.1.0`

- Install + setup made quiet and idempotent. - Expert and agent-tier registration during install no longer waits on the daemon being enabled — they are dispatch-routing config used by interactive flows too, so an enabled plugin always registers them. Expert-class selection is asked only when the experts list is still empty; on a populated list the install completes silently.
