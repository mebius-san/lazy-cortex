## Highlights since 5.15.0

- Dashboards can now tell a scheduler heartbeat from real work: a new `routine_runs_total` counter increments only on non-idle ticks (an inbox / md-scan / git poll that matched nothing counts as a tick but not as a run), so a routine polling every minute no longer looks like a thousand daily launches.
- md-scan routine globs accept recursive `**` patterns, so one path entry can cover a whole subtree instead of enumerating every nesting depth by hand.
