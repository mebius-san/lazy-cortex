# lazycortex-python — unpublished changes

**Status**: changed — `plugin.json` is at `1.9.0` while the public mirror still ships `1.8.1`. Pending publish.

## Highlights since 1.8.1

- `chk-py` / `tst-py` now source an optional project env script (`python.env_source` in `.claude/lazy.settings.json`) after the venv is active — so a repo that bootstraps secret paths or provider credentials from its own shell wrapper runs tests and checkers fully configured instead of half-set-up. A missing configured script aborts the runner instead of running blind.
- `/lazy-python.install` records `python.env_source` when the repo ships a recognised bootstrap script (`cli/env`, `.env.sh`, `scripts/env.sh`), asking only when more than one candidate is present and never overwriting an existing value.
- The style rule's Verification Order now yields to a project's own test/check runner when one is declared, replacing `chk-py` / `tst-py` at every step.
